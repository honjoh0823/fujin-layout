# Governance

Fujin Layout is currently maintained by a single primary maintainer:

- 本城 靖大 / honjoh0823

The maintainer is responsible for:

- The core layout design.
- The AutoHotkey implementation.
- Release decisions.
- Reviewing issues, benchmark reports, and accessibility feedback.
- Deciding whether proposed changes preserve the project's accessibility goal.

## Project Direction

The project prioritizes:

1. One-handed Japanese input accessibility.
2. Usability on ordinary keyboards without custom hardware.
3. Clear documentation for non-technical users.
4. Reproducible benchmark data.
5. Cross-platform portability where feasible.

## Contributions

Contributions are welcome through issues and pull requests. Because the layout
is still early-stage, major design changes should start as issues or benchmark
reports before implementation.
