# Contributing

Fujin Layout welcomes bug reports, accessibility feedback, benchmark data, and
implementation improvements.

## Good Contributions

- Reports from users who type with one hand because of disability, injury, or
  temporary constraints.
- Keyboard compatibility issues on Windows.
- Suggestions that make installation easier for non-technical users.
- Benchmark results with the practice time, test text, keyboard model, and
  typing score included.
- Pull requests that improve the AutoHotkey v2 script while preserving the
  core layout concept.

## Reporting Issues

When opening an issue, include:

- Windows version.
- AutoHotkey version.
- Keyboard layout and keyboard model if relevant.
- Steps to reproduce the problem.
- Expected behavior and actual behavior.
