# Roadmap

Fujin Layout is maintained as an open-source accessibility and input-method
research project. The goal is to make high-speed Japanese text input possible
with one hand on ordinary keyboards.

## Near Term

- Improve the AutoHotkey v2 implementation and reduce edge-case input bugs.
- Add clearer setup instructions for Windows users.
- Publish reproducible benchmark notes and training examples.
- Collect feedback from users with one-handed input needs.
- Add diagrams for the main layer, vowel layer, and CapsLock layer.

## Medium Term

- Build an installer or packaged release for non-technical users.
- Add configuration options for keyboard size, handedness, and modifier keys.
- Explore right-hand-only and mirrored variants.
- Add automated checks for layout mapping regressions.

## Long Term

- Port the layout to macOS and Linux input frameworks where feasible.
- Compare Fujin against existing one-handed and accessibility-oriented input
  systems.
- Document a public research dataset for Japanese one-handed typing practice.
